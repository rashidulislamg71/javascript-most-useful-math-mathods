
### JavaScript Most useful Math Methods ###
-------------------------------------------

***Math.random()***
--Randomly সংখ্যা তৈরি করতে ব্যবহার হয় Math.random()। এটি ০ থেকে ১ এর মধ্যে (১ ছাড়া) একটি এলোমেলো (দশমিক) সংখ্যা দেয়।

For Example:

    const randomNumber = Math.random();
    console.log(randomNumber) 

Explanation:
    এখানে প্রতিবার refresh / run করলে ০ থেকে ১ এর ছোট একটি নতুন random সংখ্যা দেখাবে। কিন্তু ১ কখনোই আসবে না

-----------------------------------------------------------------------------------------------

***Math.max()***
    Math.max() একাধিক সংখ্যার মধ্যে থেকে সবচেয়ে বড় সংখ্যা বের করে দেয়।

For Example:

    const maxNumber = Math.max(12, 32, 4, 53);
    console.log(maxNumber);

Explanation:
    এখানে 12, 32, 4, 53 — এই সংখ্যাগুলোর মধ্যে
    সবচেয়ে বড় সংখ্যা হলো 53।
    তাই আউটপুট হবে: 53
    
-----------------------------------------------------------------------------------------------

***Math.min() ***
    Math.min() একাধিক সংখ্যার মধ্যে থেকে সবচেয়ে ছোট সংখ্যা বের করে দেয়।

For Example:

    const maxNumber = Math.max(12, 32, 4, 53);
    console.log(maxNumber);

Explanation:
    এখানে 12, 32, 4, 53 — এই সংখ্যাগুলোর মধ্যে
    সবচেয়ে ছোট সংখ্যা হলো 4।
    তাই আউটপুট হবে: 4
    
-----------------------------------------------------------------------------------------------

***Math.floor() ***
    Math.floor() কোনো দশমিক সংখ্যাকে নিচের পূর্ণসংখ্যায় নামিয়ে আনে।

For Example:

    const number = Math.floor(5.9);
    console.log(number);

Explanation:
    এখানে 5.9 এর নিচের পূর্ণসংখ্যা হলো 5।
    তাই আউটপুট হবে: 5
    
-----------------------------------------------------------------------------------------------

***Math.ceil() ***
    Math.ceil() যেকোনো দশমিক সংখ্যাকে উপরের পূর্ণসংখ্যায় তুলে নেয়।

For Example:

    const number = Math.ceil(5.1);
    console.log(number);

Explanation:
    এখানে 5.1 এর উপরের পূর্ণসংখ্যা হলো 6।
    তাই আউটপুট হবে: 6
    
-----------------------------------------------------------------------------------------------

***Math.trunc() ***
    Math.trunc() কোনো দশমিক সংখ্যা থেকে দশমিকের পরের অংশটুকু বাদ দিয়ে শুধু পূর্ণসংখ্যাটি রাখে। এটি floor বা ceil এর মতো হিসেব করে না, শুধু দশমিক অংশটা কেটে ফেলে।

For Example:

    const number = Math.trunc(4.9);
    console.log(number); // আউটপুট: 4

Explanation: 
    এখানে 4.9 থেকে দশমিকের পরের অংশ (.9) বাদ দিয়ে দেওয়া হয়েছে।
    তাই আউটপুট 4
    
-----------------------------------------------------------------------------------------------

***Math.round() ***
    Math.round() যে কোনো সংখ্যাকে নিকটতম পূর্ণসংখ্যায় round করে। 

For Example:

    const number = Math.round(4.6);
    console.log(number);

    const number2 = Math.round(4.3);
    console.log(number2);

Explanation:
    এখানে number 4.6 সংখ্যা 5 এর কাছাকাছি।
    তাই আউটপুট হবে: 5
    আবার number2 4.3 সংখ্যা 4 এর কাছাকাছি।
    তাই আউটপুট হবে: 4
    
-----------------------------------------------------------------------------------------------

***Math.abs() ***
    Math.abs() যেকোন সংখ্যা কে ধনাত্মক (absolute) সংখ্যায় রুপান্তর করে।

For Example:

    const number = Math.abs(-20);
    console.log(number);

Explanation:
    এখানে -20 এর ধনাত্মক মান হলো 20।
    তাই আউটপুট হবে: 20
    
-----------------------------------------------------------------------------------------------

***Math.pow() ***
    Math.pow() যে কোনো সংখ্যার power (ঘাত) বের করতে ব্যবহার হয়।
    এটি দুইটি প্যারামিটার নেয়—
    প্রথমটি হলো মূল সংখ্যা, আর দ্বিতীয়টি হলো কত power (ঘাত) দিতে হবে।

For Example:

    const number = Math.pow(2, 4);
    console.log(number);

Explanation:
    এখানে 2 এর power 4 → 2 × 2 × 2 × 2 = 16।
    তাই আউটপুট হবে: 16
    
-----------------------------------------------------------------------------------------------

***Math.sqrt() ***
    Math.sqrt() কোনো সংখ্যার square root বের করে।

For Example:

    const number = Math.sqrt(36);
    console.log(number);

Explanation:
    এখানে 36 এর square root হলো 6।
    তাই আউটপুট হবে: 6
